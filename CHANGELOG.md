# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),

## [1.0.0] - 2024-11-05

### Added
- Initial release
- React + TypeScript + Vite setup
- TMDB API integration for 75K+ movies and TV shows
- Netflix-inspired UI/UX design
- Search functionality with actor search
- Actor profile pages with biography
- Sports streaming page
- Progress tracking with localStorage
- Responsive design for all devices
- Auto-setup script for easy installation
- API key warning banner

### Features
- Home page with curated content sections
- Movies and TV shows pages
- Advanced search with filters
- Live sports streaming
- Top 10 trending content
- Actor collections
- Franchise collections
- Continue watching
- My List functionality
